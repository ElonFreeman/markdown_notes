## 霍夫曼树节点数与叶子数的关系：2n-1 的数学证明

### 基本概念

- 霍夫曼树是一棵**二叉树**

- 具有 **n 个叶子节点**（待编码的字符）

- 所有内部节点的**度为2**（都有左右两个孩子）

### 证明方法

#### 方法一：度数关系法（最简洁）

设：

- **n** = 叶子节点数

- **m** = 内部节点数

- **N** = 总节点数 = n + m

对于任意二叉树，存在以下关系：

**1. 边数关系**

- 每个节点（除根节点外）都有一条入边

- 总边数 = 总节点数 - 1 = N - 1

**2. 度数关系**

- 每个叶子节点：出度为0

- 每个内部节点：出度为2（霍夫曼树的内部节点都有2个孩子）

总出度 = 2m

**3. 出入度相等**  
总入度 = 总出度

text

N - 1 = 2m

将 N = n + m 代入：

text

(n + m) - 1 = 2m
n + m - 1 = 2m
n - 1 = m

所以：

text

m = n - 1
N = n + m = n + (n - 1) = 2n - 1

**证毕** ✓
